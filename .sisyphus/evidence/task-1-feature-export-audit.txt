### 1. MISSING EXPORT: `cardkit` feature (CRITICAL)
- **Location**: `Cargo.toml:250` defines `cardkit = ["client", "openlark-cardkit"]`
- **Missing**: `src/lib.rs` has NO `#[cfg(feature = "cardkit")] pub use openlark_cardkit;`
- **Impact**: Users cannot access cardkit module even when feature is enabled
- **RC Action Required**: Add the missing export to `src/lib.rs`

### 2. MISSING PRELUDE EXPORT (CRITICAL)

- **Impact**: README examples won't compile
