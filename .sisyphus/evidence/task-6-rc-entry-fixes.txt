# Task 6 - RC Entry Consistency Fixes

## Summary of Issues (from Task 1 Audit)

### Critical Issue 1: Missing `cardkit` Export
- **File**: `src/lib.rs`
- **Missing**: `#[cfg(feature = "cardkit")] pub use openlark_cardkit;`
- **Impact**: Users cannot access cardkit module even with feature enabled

### Critical Issue 2: Missing `prelude` Module
- **File**: `src/lib.rs`
- **Missing**: `pub mod prelude` or re-export
- **Impact**: README examples using `use open_lark::prelude::*;` won't compile
